# Code Quality Analyzer

An AI-powered Chrome Extension that helps developers analyze code quality, identify issues, and receive optimization suggestions using Groq AI.

## Features

* AI-powered code analysis
* Code quality assessment
* Optimization recommendations
* Complexity insights
* Best practice suggestions
* Secure Groq API integration

## Tech Stack

* Chrome Extension (Manifest V3)
* JavaScript, HTML, CSS
* Node.js, Express.js
* Groq API
* Render

## Setup

1. Clone the repository
2. Install backend dependencies (`npm install`)
3. Add your Groq API key in `.env`
4. Start the backend (`npm start`)
5. Load the `extension` folder via `chrome://extensions`
