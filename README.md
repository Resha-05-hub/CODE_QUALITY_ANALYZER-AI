#  Code Quality Analyzer (AI-Powered Chrome Extension)

An AI-powered Chrome Extension that helps developers analyze code quality, detect issues, and get optimization suggestions using **Groq AI**.

---

##  Overview

Code Quality Analyzer runs inside the browser and provides real-time AI feedback on code.  
It helps improve code structure, performance, and best practices using intelligent analysis.

---

##  Features

-  AI-powered code analysis using Groq API  
-  Detects bugs and bad practices  
-  Performance and complexity insights  
-  Optimization suggestions  
-  Code quality feedback  
-  Secure API integration  

---

## Tech Stack

- Chrome Extension (Manifest V3)  
- HTML, CSS, JavaScript  
- Node.js & Express.js  
- Groq AI API  
- Render Hosting  

---

## Setup Instructions

### 1. Clone Repository
```bash
git clone https://github.com/your-username/code-quality-analyzer.git
cd code-quality-analyzer
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Create `.env` File
```env
GROQ_API_KEY=your_api_key_here
PORT=5000
```

### 4. Run Backend
```bash
npm start
```

### 5. Load Chrome Extension
- Open `chrome://extensions/`
- Enable **Developer Mode**
- Click **Load unpacked**
- Select `extension/` folder
